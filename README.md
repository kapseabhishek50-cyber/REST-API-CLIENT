HTTP COMMUNICATION:
THE PROGRAM UTILIZES THE JAVA.NET.HTTP.HTTPCLIENT API. THIS IS THE MODERN STANDARD IN 2026 FOR HANDLING ASYNCHRONOUS AND SYNCHRONOUS REQUESTS. IT SUPPORTS HTTP/2 AND IS DESIGNED TO BE MORE EFFICIENT THAN THE LEGACY HTTPURLCONNECTION.
REQUEST CONFIGURATION:
A HTTPREQUEST OBJECT IS CONSTRUCTED USING A BUILDER PATTERN. IT DEFINES THE TARGET URL (OPEN-METEO API), THE METHOD (GET), AND A CONNECTION TIMEOUT TO PREVENT THE PROGRAM FROM HANGING IF THE SERVER IS UNRESPONSIVE.
